# Booking Service Backend

## Setup

```bash
npm install
npm run generate:api-types
npm run dev
```

Server starts on http://localhost:3000.

## Configuration

| Variable | Default  | Description       |
|----------|----------|-------------------|
| PORT     | 3000     | Server port       |
| HOST     | 0.0.0.0  | Bind address      |

See `.env.example`.
